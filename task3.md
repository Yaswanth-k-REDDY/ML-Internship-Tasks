# Task 3: Linear Regression

## Workflow

### Step 1: Import Libraries
Imported Pandas, NumPy, Matplotlib and Scikit-Learn libraries required for data manipulation, visualization and machine learning.

### Step 2: Load Dataset
Loaded the Housing dataset using Pandas DataFrame.

### Step 3: Data Preprocessing
Converted categorical variables into numerical format using One-Hot Encoding.

### Step 4: Feature Selection
Separated:
- Features (X)
- Target Variable (y = price)

### Step 5: Train-Test Split
Split the dataset into:
- 80% Training Data
- 20% Testing Data

### Step 6: Model Training
Created and trained a Linear Regression model using Scikit-Learn.

### Step 7: Prediction
Generated house price predictions on unseen test data.

### Step 8: Evaluation
Measured performance using:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

### Step 9: Visualization
Plotted Actual vs Predicted House Prices to analyze model performance.

## Output
The model successfully predicts house prices based on multiple housing features and provides performance metrics for evaluation.

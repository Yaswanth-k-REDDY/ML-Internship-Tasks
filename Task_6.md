# K-Nearest Neighbors (KNN) Classification

## Objective
Implement the KNN algorithm for classification using the Iris dataset.

## Tools Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Dataset
Iris Dataset

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target:
- Setosa
- Versicolor
- Virginica

## Steps
1. Load dataset
2. Normalize features using StandardScaler
3. Split into training and testing sets
4. Train KNN classifier
5. Experiment with different K values
6. Evaluate accuracy
7. Generate confusion matrix
8. Visualize decision boundaries

## Results
- Best K value selected based on highest accuracy.
- Model evaluated using:
  - Accuracy Score
  - Confusion Matrix

## Concepts Learned
- Distance-based learning
- Feature scaling importance
- Hyperparameter tuning (K value)
- Classification evaluation metrics
- Decision boundary visualization

# Task 2: Exploratory Data Analysis (EDA)

## Objective
The objective of this task is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data, identify patterns, detect anomalies, and gain insights through statistical summaries and visualizations.

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Titanic Dataset

## Tasks Performed
- Loaded and explored the dataset.
- Generated summary statistics such as mean, median, standard deviation, minimum, and maximum values.
- Checked for missing values in the dataset.
- Created histograms to understand the distribution of numerical features.
- Created boxplots to identify outliers.
- Generated a correlation matrix and heatmap to analyse relationships between variables.
- Created pairplots for feature comparison and visualisation.
- Analysed survival patterns based on gender and passenger class.
- Calculated skewness of numerical features.

## Key Insights
- Female passengers had a higher survival rate compared to male passengers.
- First-class passengers had better survival chances than passengers in lower classes.
- Fare distribution contained several outliers.
- Age distribution was concentrated among young and middle-aged passengers.
- Some features showed skewed distributions and required further analysis.

## Outcome
Successfully performed Exploratory Data Analysis on the Titanic dataset using statistical methods and visualisations. The analysis helped in understanding the dataset structure, identifying trends, detecting outliers, and extracting meaningful insights that can support future machine learning tasks.
